# Installing
First of all, clone the repo in your machine
```git
git clone https://github.com/dgzdev/fvcker-tool.git
```
Go to the directory of the repo
```cmd
cd fvcker-tool
```
Give permission for the script execution
```sh
chmod +x fvcker.sh
```
Now, run the script and be free.
```sh
./fvcker.sh
```
